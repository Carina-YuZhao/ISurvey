## Overview

**ISurvey** is an R toolkit for the analysis of interval-valued survey data. It is intended for studies in which respondents provide interval-valued answers, and for comparative workflows involving both interval-valued and conventional single-valued survey responses.

Usage of the toolkit should follow the methodology and reporting context described in the associated paper. If you use **ISurvey** in academic work, please cite: **Yu Zhao, Christian Wagner, Brendan Ryan, and Direnc Pekaslan, *ISurvey: An R Toolkit for Interval-Valued Survey Data Analysis*, IEEE International Conference on Fuzzy Systems (FUZZ-IEEE), 2026.**
